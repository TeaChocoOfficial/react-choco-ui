# @teachoco-official/react-choco-ui

A React component library for building modern UI, inspired by Tailwind.

## Status

**Beta Version**  
This library is currently in beta and is not yet fully stable or complete. Features may change, and bugs may be present. Use with caution in production environments.

## Overview

`@teachoco-official/react-choco-ui` is a lightweight and customizable UI component library for React applications. It draws heavy inspiration from [Material-UI (MUI)](https://mui.com/), offering a similar design philosophy but with a unique flavor tailored for specific use cases.

## Installation

To install the library, run:

```bash
npm install @teachoco-official/react-choco-ui
```

or with Yarn:

```bash
yarn add @teachoco-official/react-choco-ui
```

or with pnpm:

```bash
pnpm add @teachoco-official/react-choco-ui
```
